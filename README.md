# SICP-learn: Foundations of Computational Logic

A comprehensive repository dedicated to the study of **"Structure and Interpretation of Computer Programs" (SICP)**. This project serves as a structured laboratory for mastering computer science fundamentals, procedural abstraction, and linguistic design.

## 🎯 Project Goals
* Master the art of building complex systems from simple, robust components.
* Explore different evaluation models and computational processes.
* Develop a deep understanding of functional programming and linguistic abstraction.

## 🛠 Tech Stack
* **Language:** Scheme (Lisp dialect)
* **Environment:** MIT/GNU Scheme
* **Editor:** VS Code
* **Extension:** `scheme-vscode` (by Alen Huang)

## 📁 Repository Structure
The repository is organized by chapters as defined in the SICP curriculum:

* **[Chapter 1](./chapter-1/):** Building Abstractions with Procedures.
    * Focus: Expressions, Substitution Model, Higher-Order Procedures.
* **Chapter 2 (Planned):** Building Abstractions with Data.
* **Chapter 3 (Planned):** Modularity, Objects, and State.
* **Chapter 4 (Planned):** Metalinguistic Abstraction.
* **Chapter 5 (Planned):** Computing with Register Machines.
